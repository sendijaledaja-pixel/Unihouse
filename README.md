# Unihouse Plānotājs

Personīgais darbu, grafika un tikšanos plānotājs.

## Izvietošana (Vercel)

1. Augšupielādē šo repo uz GitHub
2. Vercel → New Project → Import repo
3. Framework: **Vite** (automātiski nosaka)
4. Deploy → gatavs!

## Lokāla palaišana

```bash
npm install
npm run dev
```

## PWA (Add to Home Screen)

Pēc deploy → atver linku telefona pārlūkā:
- **iPhone Safari**: Share → Add to Home Screen
- **Android Chrome**: ⋮ → Add to Home Screen
